Recomendaciones de navegación

Bienvenido al módulo, para que vivas una experiencia de aprendizaje ágil, toma en cuenta las siguientes recomendaciones:

  • Asegúrate de tener una conexión de internet estable.
  • Asegura un espacio adecuado para que puedas centrar tu atención unos minutos a este contenido.
  • Revisa tu pantalla, para que conozcas cada elemento y te familiarices con ella.
  • Ten a la mano un cuaderno de notas para registras la información.

¡Recuerda que una evaluación de aprendizaje te espera al final de cada módulo!
